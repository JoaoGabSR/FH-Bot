# FH Bot

O **FH** é um bot para Discord criado a partir da ideia de um amigo que queria montar um comboio de corredores, junto com outro amigo que adora transformar qualquer ideia em código e linhas coloridas.

O projeto surgiu da necessidade de criar um bot com funcionalidades voltadas para a comunidade de jogadores de **Forza Horizon**, especialmente com a chegada do novo título da franquia: **Forza Horizon 6**, que, segundo os rumores e expectativas da comunidade, se passará no belíssimo cenário do **Japão**.

A proposta inicial do bot é ajudar na organização de **comboios**, facilitando a comunicação entre os membros do servidor e a confirmação de presença em eventos dentro da comunidade.

## Objetivo inicial

O foco da primeira versão do bot é permitir que administradores da comunidade criem convites para comboios, registrem respostas dos participantes e organizem automaticamente listas de presença para eventos futuros.

## Futuras funcionalidades

Novas funcionalidades serão adicionadas conforme a ideia do projeto for amadurecendo e evoluindo junto com as necessidades da comunidade.

Este é o meu **primeiro bot para Discord**, então ainda tenho muito a aprender ao longo do desenvolvimento.  
Se você quiser contribuir com ideias, melhorias ou implementações, será muito bem-vindo. Toda ajuda e compartilhamento de conhecimento são sempre valiosos.
